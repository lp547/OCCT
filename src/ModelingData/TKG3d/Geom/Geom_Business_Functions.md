# OCCT Geom 包 - 业务功能与概览

## 1. 简介
**Geom** 包是 Open CASCADE Technology (OCCT) **ModelingData** 模块的基础组件，具体位于 **TKG3d** (Toolkit Geometry 3D) 中。它提供了在 3D 欧几里得空间中定义、操作和查询几何对象所需的数学数据结构和算法。

## 2. 核心业务功能
`Geom` 包的主要业务功能是通过提供强大的数学形状描述，支持 **计算机辅助设计 (CAD)**、**制造 (CAM)** 和 **工程 (CAE)** 工作流。

### 2.1. 几何表示
该包实现了符合标准 (ISO 10303-42 / STEP) 的定义：
*   **基本曲线**：直线 (Lines)、圆 (Circles)、椭圆 (Ellipses)、双曲线 (Hyperbolas)、抛物线 (Parabolas)。
*   **自由形式曲线**：贝塞尔曲线 (Bézier curves)、B 样条曲线 (B-Spline curves / NURBS)。
*   **基本曲面**：平面 (Planes)、圆柱面 (Cylinders)、圆锥面 (Cones)、球面 (Spheres)、环面 (Tori)。
*   **自由形式曲面**：贝塞尔曲面 (Bézier surfaces)、B 样条曲面 (B-Spline surfaces / NURBS)。
*   **扫掠曲面**：旋转面 (Surfaces of Revolution)、线性拉伸面 (Linear Extrusion)。
*   **裁剪/有界几何**：由参数范围限定的曲线和曲面。

### 2.2. 参数化建模
`Geom` 中的所有几何体都是**参数化**的。这意味着形状由方程定义：曲线为 $C(u)$，曲面为 $S(u,v)$，其中 $u$ 和 $v$ 是实数参数。
*   **业务价值**：允许精确计算几何体上任意位置的点、导数（切线、法线）和曲率，这对于求交算法、网格划分和数控 (NC) 路径生成至关重要。

### 2.3. 几何变换
该包支持非变形变换（刚体运动 + 缩放）：
*   平移 (Translation)
*   旋转 (Rotation)
*   对称/镜像 (Symmetry / Mirror)
*   缩放 (Scaling)
*   复杂变换（基于矩阵，通过 `gp_Trsf`）

### 2.4. 几何连续性管理
确保形状满足高质量曲面建模（A 类曲面）和制造所需的连续性要求（$C^0, C^1, C^2, ...$ 或 $G^1, G^2, ...$）。

## 3. 使用场景

### 3.1. CAD 建模操作
*   用作拓扑边和面（`TopoDS_Edge`, `TopoDS_Face`）的底层定义。
*   例如：CAD 模型中的圆柱面指向一个 `Geom_CylindricalSurface` 实例。

### 3.2. 数据交换
*   作为导入/导出标准格式（如 **STEP** 和 **IGES**）的原生数据结构，这些格式共享类似的几何定义（NURBS，解析曲面）。

### 3.3. 分析与可视化
*   提供必要的数学评估方法（`D0`, `D1`, `D2`）以生成显示网格（三角剖分）并执行物理分析。

## 4. 关键差异化优势
*   **精度**：使用 `Standard_Real`（双精度）进行高精度工程计算。
*   **持久性**：对象可以被序列化和反序列化。
*   **内存管理**：使用引用计数机制（`Handle`）在多个拓扑实体之间安全地共享几何数据（例如，两个面共享同一个底层曲面）。
