
以下是 `src/ModelingData/TKBRep/BRep` 目录下各文件的详细功能解释。该目录主要包含 Open CASCADE Technology (OCCT) 中边界表示 (BRep) 数据结构的底层实现。

### BRep_Builder (构建器)
**BRep_Builder.hxx / .cxx / .lxx**
该类提供了一个构建 BRep 拓扑结构（如 Vertex, Edge, Face 等）的框架。它继承自 `TopoDS_Builder`，不仅能够创建基本的拓扑元素，还提供了高级的公差控制功能。它用于将几何对象（如 `Geom_Surface`, `Geom_Curve`, `Poly_Triangulation` 等）关联到拓扑形状上，是创建和修改 BRep 模型的核心工具。

### BRep_Tool (工具类)
**BRep_Tool.hxx / .cxx**
这是一个静态工具类，用于访问 BRep 形状（Shapes）中的几何数据。它提供了从 `TopoDS_Shape` 中提取几何信息的方法，例如获取 `TopoDS_Face` 的曲面、`TopoDS_Edge` 的曲线、`TopoDS_Vertex` 的点坐标以及它们的公差。它是开发者读取和查询 BRep 模型几何属性最常用的接口。

### BRep_TFace (面数据结构)
**BRep_TFace.hxx / .cxx**
该类是 `TopoDS_TFace` 的 BRep 具体实现，用于存储面的几何数据。它包含了一个几何曲面 (`Geom_Surface`)、位置 (`TopLoc_Location`)、公差 (`Tolerance`) 以及可选的三角剖分数据 (`Poly_Triangulation`)。它还包含一个 `NaturalRestriction` 标志，用于指示面的边界是否即为参数空间的边界。

### BRep_TEdge (边数据结构)
**BRep_TEdge.hxx / .cxx / .lxx**
该类是 `TopoDS_TEdge` 的 BRep 具体实现，用于存储边的几何数据。它包含公差、一些状态标志（如 SameParameter, SameRange, Degenerated）以及一个曲线表示列表 (`BRep_ListOfCurveRepresentation`)。这个列表可以包含该边在不同面上的参数曲线 (pcurves) 或其 3D 曲线。

### BRep_TVertex (顶点数据结构)
**BRep_TVertex.hxx / .cxx / .lxx**
该类是 `TopoDS_TVertex` 的 BRep 具体实现，用于存储顶点的几何数据。它主要包含一个 3D 点 (`gp_Pnt`)、公差以及一个点表示列表 (`BRep_ListOfPointRepresentation`)。该列表用于存储顶点在边或面上的参数信息。

### BRep_CurveRepresentation (曲线表示基类)
**BRep_CurveRepresentation.hxx / .cxx / .lxx**
这是所有边（Edge）几何表示的抽象基类。由于一条边可能有多种几何形式（例如 3D 曲线、面上的 2D 曲线、多边形等），该类及其派生类用于统一管理这些不同的表示形式，并存储它们的位置信息。

### BRep_GCurve (几何曲线基类)
**BRep_GCurve.hxx / .cxx / .lxx**
这是所有基于连续几何曲线（而非离散多边形）表示的基类，继承自 `BRep_CurveRepresentation`。它增加了参数范围（First 和 Last）的概念，用于定义边在曲线上的有效区间。

### BRep_Curve3D (3D 曲线)
**BRep_Curve3D.hxx / .cxx**
该类继承自 `BRep_GCurve`，用于表示一条边在 3D 空间中的几何形状。它持有一个 `Geom_Curve` 对象，代表了边的实际 3D 曲线几何。

### BRep_CurveOnSurface (面上曲线)
**BRep_CurveOnSurface.hxx / .cxx / .lxx**
该类继承自 `BRep_GCurve`，用于表示一条边在一个面上的参数曲线（PCurve）。它持有一个 `Geom2d_Curve` 和对应的 `Geom_Surface`，用于定义边在特定曲面参数空间中的轨迹。

### BRep_CurveOnClosedSurface (闭合面上曲线)
**BRep_CurveOnClosedSurface.hxx / .cxx / .lxx**
该类继承自 `BRep_CurveOnSurface`，专门用于处理闭合曲面（如圆柱、球）上的边。它存储了两条 2D 曲线（PCurve1 和 PCurve2），分别对应曲面闭合缝隙两侧的参数定义，确保跨越闭合边界时的连续性。

### BRep_CurveOn2Surfaces (两面上曲线/正则性)
**BRep_CurveOn2Surfaces.hxx / .cxx**
该类用于定义两个曲面之间的几何连续性（Regularity）。它通常不包含显式的曲线几何，而是存储两个曲面及它们之间的连续性级别（如 G1, G2 等），常用于处理类似圆角或缝合边的平滑过渡信息。

### BRep_Polygon3D (3D 多边形)
**BRep_Polygon3D.hxx / .cxx**
该类继承自 `BRep_CurveRepresentation`，用于边的离散表示。它持有一个 `Poly_Polygon3D` 对象，即一系列 3D 点，通常用于可视化或没有解析几何曲线的情况。

### BRep_PolygonOnSurface (面上多边形)
**BRep_PolygonOnSurface.hxx / .cxx**
该类用于表示边在曲面参数空间中的离散形式。它持有一个 `Poly_Polygon2D`（一系列 UV 点）和一个 `Geom_Surface`，常用于网格化（Meshing）后的数据存储。

### BRep_PolygonOnClosedSurface (闭合面上多边形)
**BRep_PolygonOnClosedSurface.hxx / .cxx**
类似于 `BRep_CurveOnClosedSurface`，该类用于闭合曲面上的离散边表示。它存储了两个 `Poly_Polygon2D` 对象，分别对应闭合曲面缝隙两侧的参数空间多边形。

### BRep_PolygonOnTriangulation (三角网上的多边形)
**BRep_PolygonOnTriangulation.hxx / .cxx**
该类用于表示基于三角网格（Triangulation）的边。它存储了一个节点索引数组 (`Poly_PolygonOnTriangulation`)，这些索引指向所属面的 `Poly_Triangulation` 中的节点。这是现代 OCCT 显示模块中常用的边的显示数据格式。

### BRep_PolygonOnClosedTriangulation (闭合三角网上的多边形)
**BRep_PolygonOnClosedTriangulation.hxx / .cxx**
该类是 `BRep_PolygonOnTriangulation` 的扩展，用于闭合曲面的三角网格。它存储了两组节点索引，分别对应闭合边界两侧的离散表示。

### BRep_PointRepresentation (点表示基类)
**BRep_PointRepresentation.hxx / .cxx / .lxx**
这是顶点（Vertex）几何表示的抽象基类。顶点除了有 3D 坐标外，还可能需要参数信息（例如在某条边上的参数值）。该类及其子类用于存储这些附加的参数信息。

### BRep_PointOnCurve (曲线上点)
**BRep_PointOnCurve.hxx / .cxx**
该类表示一个顶点位于某条 3D 曲线 (`Geom_Curve`) 上。它存储了一个参数值 (Parameter)，定义了顶点在曲线上的位置。

### BRep_PointsOnSurface (面上点基类)
**BRep_PointsOnSurface.hxx / .cxx**
这是位于曲面上的点的表示基类，继承自 `BRep_PointRepresentation`。它持有一个 `Geom_Surface` 对象。

### BRep_PointOnCurveOnSurface (面上曲线上的点)
**BRep_PointOnCurveOnSurface.hxx / .cxx**
该类表示一个顶点位于曲面上的某条 2D 曲线 (`Geom2d_Curve`) 上。它存储了一个参数值，用于定义顶点在该 2D 曲线上的位置。

### BRep_PointOnSurface (面上点)
**BRep_PointOnSurface.hxx / .cxx**
该类表示一个顶点直接位于曲面上。它存储了两个参数值 (U, V)，定义了顶点在曲面参数空间中的确切坐标。

### 辅助列表类
**BRep_ListOfCurveRepresentation.hxx**
**BRep_ListOfPointRepresentation.hxx**
**BRep_ListIteratorOf...**
这些是标准的集合类，分别用于存储 `BRep_CurveRepresentation` 和 `BRep_PointRepresentation` 对象的列表。它们被 `BRep_TEdge` 和 `BRep_TVertex` 用来管理其所有的几何表示形式。

### 其他辅助类
**BRep_EdgeType.hxx**
该枚举定义了边的类型（如线性、曲线、多边形等），用于在不同的算法中进行区分。

**BRep_EdgeType.hxx**
该枚举定义了顶点的类型（如点、曲线上的点、面上的点等），用于在不同的算法中进行区分。
